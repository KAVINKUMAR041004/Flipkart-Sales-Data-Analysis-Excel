# Flipkart Sales Data Analysis

This project analyzes Flipkart e-commerce sales data using Excel.

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Data Visualization

## Project Tasks
- Cleaned and structured raw Flipkart dataset
- Analyzed product categories and pricing trends
- Identified top-selling products
- Created pivot tables for sales insights
- Built charts to visualize sales performance

## Insights
- Top performing product categories
- Pricing distribution
- Product rating trends
- Customer preference patterns

## Files
FLIPKART.xlsx – Dataset and analysis
flipkart.csv – Raw dataset
